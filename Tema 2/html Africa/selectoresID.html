En ocasiones, es neceario aplicar estilo CSS a un único elemento de la página

Aunque puede utilizarse un selector de clase para aplicar estilos a un único elemento, existe otro selector más eficiente.

El selector Id permite seleccionar un elemento de la página a través del valor de su atributo id

Este tipo de selectores sólo seleccionan Un eleemento de la página porque el valor del atriburo Id no se puede repetir en dos elementos diferentes de una misma página.


La recomendación general es utilizar el selectoresid cuando se quiere aplicar un estilo a un solo elemento específico de la página y utilizar el selector de clase cuando se quiera aplicar un estilo a varios diferentes.
